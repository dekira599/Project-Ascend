# PRODYGY - Catálogo Web de Productos Tecnológicos

Proyecto académico de desarrollo web enfocado en la creación de un catálogo digital de productos tecnológicos con funcionalidades de búsqueda, filtrado, carrito de compras y control básico de stock.

## Descripción

PRODYGY es una aplicación web desarrollada como proyecto académico para simular una tienda en línea de productos tecnológicos.  
El sistema permite visualizar productos, consultar detalles, agregarlos al carrito, calcular el total de compra y generar un mensaje de pedido mediante WhatsApp.

El proyecto fue desarrollado con tecnologías web básicas, aplicando lógica de programación, manipulación del DOM, almacenamiento local y organización de datos mediante JavaScript.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript
- JSON
- Local Storage
- PHP básico para simulación de compra

## Funcionalidades principales

- Visualización de catálogo de productos.
- Búsqueda de productos por nombre.
- Filtrado por categorías.
- Vista detallada de cada producto.
- Carrito de compras dinámico.
- Cálculo automático del total.
- Control básico de stock.
- Eliminación de productos del carrito.
- Opción para vaciar completamente el carrito.
- Almacenamiento del carrito mediante Local Storage.
- Generación de pedido por WhatsApp.
- Simulación de actualización de stock mediante archivo PHP.
